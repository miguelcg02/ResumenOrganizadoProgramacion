# Funciones clase Math

```
.:Devuelve el número de formas de elegir k elementos de n elementos sin repetición y sin orden:.
math.comb(n, k)

.:Devuelve el número de formas de elegir k elementos de n elementos sin repetición, pero con orden:.
math.perm(n, k=None)

.:Retorna la magnitud de X pero el signo de Y:.
math.copysign(x, y)

.:Retorna el valor absoluto de x:.
math.fabs(x)

.:Retorna el factorial de x:.
math.factorial(x)

.:Retorna un flotante cercano:.
math.fsum([.1,.1,.1,.1])

.:Retorna el maximo comun divisor:.
math.gcd(x,y,z)

.:Retorna el minimo común multiplo:.
math.lcm(x,y,z)

.:Retorna el determinante:.
math.remainder(x, y)

.:Retorna la raiz cuadrada:.
math.qrt(n)

.:Retorna x sin la parte fraccional:.
math.trunc(x)

.:Retorna el elevado de X en Y:.
math.pow(x,y)

.:Retornar el logaritmo natural de x:.
math.log(x)

.:Retorna el logaritmo de la base ingresada de x:.
math.log(x[, base])

.:Retorn el logaritmo base de 10 de x:
math.log10(x)

.:Retorn la distancia euclediana de dos punto:.
math.dist(p, q)
```
