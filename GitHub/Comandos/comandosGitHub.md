# Pasos creaci칩n de un repositorio de un proyecto

```
1. Ir al repositorio y copiar link del proyecto en "code"
2. Ejecutar el comando en consola desde la carpeta donde va a clonar el repo "git clone + el link"
3. Inicializar git flow con "git flow init"
4. Crear una feature "git flow feature star nombre_rama"
5. Publicar la rama para que todos tengan acceso "git flow feature publish nombre_rama"
```
