# Comandos basicos

para ver la lista de carpetas y archivos

```
ls
```

para indicar en que directorio estas ubicado

```
pwd
```

movernos entre carpetas

```
cd blabla/blabla
```

salirnos de la carpeta

```
cd ..
```

crear carpeta

```
mkdir nombre_que_quieras
```

borrar un archivo

```
rm archivo.txt
```
