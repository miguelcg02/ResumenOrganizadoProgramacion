# Hooks

Son funciones de WordPress que nos permiten agregar códgip

- Action, agregamos código. add action
- Filter, hace lo mismo que el action, pero tiene retorno. add filter

# Manejo librerias

- Para archivos css
* wp_register_style()
* wp_enqueue_style()

- para Js
* wp_register_script()
* wp_enqueue_script()
